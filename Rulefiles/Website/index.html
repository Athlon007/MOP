MOP: Rule Files Pack

Download the newest pack of all rule files.

Last Update: 2021-04-16

Requires MOP version: 3.0 (or better)

What is rule file?

Rule files allow MOP to know, what exceptions are needed for it to
work with other mods. Thanks to them, MOP doesn't have to be updated,
in order for it to support new mods.

If you want to learn more about mod compatibility, visit this link.

How do I use this?

Paste the content of the archive into:
MSC Mods Folder -> Config -> Mod Settings -> MOP

MOP will ignore rule files that aren't needed.
